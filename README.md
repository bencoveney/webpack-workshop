# webpack-workshop

## Set up

Clone the repo:

```bash
git clone https://github.com/bencoveney/webpack-workshop.git
cd webpack-workshop
```

Install dependencies:

```bash
npm install
```

## Running the build

Run the dev build (in watch mode):

```bash
npm run dev
```

Run the production build:

```bash
npm run prod
```
